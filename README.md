# flutter_windows

一个跨平台的桌面软件，使用流行的get来管理路由，使用getx来管理状态，使用dio来管理网络请求

# 分支介绍

main分支：最简单的最通用的模板，集成了get路由管理和getx数据状态管理，还有全局loading效果，还有shared_preferences临时数据存储，
仿照前端开发localstoreage来存储数据，项目结构也是仿照前端开发结构，让前端开发人员以熟悉的方式开发flutter软件，并且以最简单最快速的方式
来开发项目。

# 目录结构

apis: api管理，  
components：组件管理，  
pages: 页面管理，  
router：路由管理，  
stores：状态管理，  
utils：工具类

# 使用的依赖

get: ^4.6.6
dio: ^5.4.0
shared_preferences: ^2.2.2
